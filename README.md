# Team-task-manager
 A Flask-based Team Task Manager web app that helps users create, assign, and track tasks efficiently. It includes login/signup authentication, task status tracking (Pending, In Progress, Done), and a clean dashboard with statistics. Designed for team productivity and internship submission.
# Team Task Manager

This is a simple web application built using Flask that helps manage team tasks in an organized way. The system allows users to create an account, log in, and manage daily tasks from a clean dashboard.

Users can add tasks, assign them to team members, set due dates, and update their status as Pending, In Progress, or Completed. The dashboard provides a clear overview of all tasks along with basic statistics to track progress easily.

The interface is designed to be clean, responsive, and easy to use. It focuses on improving team productivity by keeping all tasks in one place.

## Features
- User signup and login
- Create, update, and delete tasks
- Assign tasks to team members
- Track task status
- Simple dashboard with task summary

## Technologies Used
- Python (Flask)
- HTML, CSS
- Bootstrap
- Jinja2 Templates

## How to Run
1. Install requirements using `pip install flask`
2. Run the app using `python app.py`
3. Open browser at `http://127.0.0.1:5000`

## Purpose
This project was created for learning Flask and understanding basic web application development with user authentication and task management features.
